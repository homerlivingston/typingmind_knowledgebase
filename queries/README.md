# Standard Queries

This folder contains standard SQL queries for retrieving key business data from the Spot Pet Insurance data warehouse.

## Query Categories

| File | Description |
|------|-------------|
| `financial_metrics.sql` | GWP, revenue, premium calculations |
| `policy_metrics.sql` | Policy counts, conversions, retention |
| `channel_performance.sql` | Acquisition channel analysis |
| `claims_metrics.sql` | Claims processing and performance |
| `customer_metrics.sql` | Customer acquisition, LTV, churn |
| `product_metrics.sql` | Plan mix, wellness adoption, species breakdown |

## Usage

Each query file contains documented SQL with:
- Query description and purpose
- Required parameters (if any)
- Expected output columns
- Example use cases

## Notes

- Adjust date ranges and filters as needed for your specific analysis
- Some queries may require access to specific schemas/tables
- Contact the data team for access or schema questions
