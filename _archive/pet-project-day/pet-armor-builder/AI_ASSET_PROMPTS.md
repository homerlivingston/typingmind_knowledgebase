## üìê TECHNICAL SPECIFICATIONS

### Sizing Guidelines
When generating, request these approximate dimensions:

- **Base Pets**: 512x512px (square, centered)
- **Body Armor**: 400x500px (slightly vertical)
- **Shields**: 600x600px (large square, center transparent)
- **Helmets**: 300x300px (square)
- **Glow Effects**: 700x700px (large square, very transparent)
- **Badges**: 128x128px (small icon)

### Layering Order (Z-Index)
For proper visual stacking:
1. Glow effects (background)
2. Shield (behind pet)
3. Body armor (on pet body)
4. Pet (center)
5. Helmet (on pet head)
6. Badges (foreground)

## üõ†Ô∏è POST-GENERATION CLEANUP

After generating with AI, you may need to:

1. **Remove backgrounds** (if AI included one)
   - Use remove.bg
   - Use Photoshop "Remove Background"
   - Use GIMP with select-by-color

2. **Adjust transparency** (for shields/glows)
   - Open in image editor
   - Adjust layer opacity
   - Save as PNG with alpha channel

3. **Resize consistently**
   - Use same scale for all armor levels
   - Keep pets at same size

4. **Center assets**
   - Use canvas resize to center
   - Ensure symmetric positioning

## üí° PLATFORM-SPECIFIC TIPS

### DALL-E 3 (ChatGPT Plus)
- Works great with detailed prompts
- Good at transparent backgrounds
- May need to specify "PNG" explicitly
- Can do 1024x1024 or 1792x1024

### Midjourney
- Add `--no background` to prompts
- Use `--ar 1:1` for square images
- May need post-processing for transparency
- Great for stylistic consistency with `--sref`

### Stable Diffusion (local/free)
- Use "transparent background" in prompt
- May need ControlNet for precise positioning
- Can use same seed for consistency
- Better control over output
